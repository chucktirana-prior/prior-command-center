# Prior Uploader Template

Copy this into a Google Doc or Word doc and replace the placeholder text.

Important:
- Bold labels are okay and preferred.
- Keep one label per line when possible.
- Do not combine labels on the same line.
- `Keep reading:` is metadata only.
- Put `Article body:` on its own line.
- Start the article body immediately after `Article body:`.
- Put inline body figures directly inside the article body using `<figure>` blocks.

```text

By: [Author name]
```
